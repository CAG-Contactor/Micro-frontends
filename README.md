Kompetensdag Micro-frontends
============================
Detta projekt innehåller de labbar vi kör under kompetensdagen för Micro Frontends.

Miljö
-----
Följande måste installeras för att köra labbarna:
- NodeJS >= 14
- JDK >= 11
- Maven >= 3
